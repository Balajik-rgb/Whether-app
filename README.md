Weather App Using HTML, CSS, and JavaScript

Project Overview :-

This Weather App is a web-based application developed using HTML, CSS, and JavaScript. The application allows users to check real-time weather information for any city by entering the city name.
The main purpose of this project is to understand how front-end technologies work together and how to fetch live data from an external API.

Technologies Used :-
HTML
Used to create the structure of the web page and input fields.
CSS
Used to design the user interface, improve layout, and make the application responsive for different devices.
JavaScript
Used to fetch weather data from an API, handle user input, and dynamically update the content on the page.
* Features : -
  
Search weather by city name
Displays temperature, humidity, and weather condition
Real-time data fetching using API
Responsive design for mobile and desktop
User-friendly interface

*How It Works :-

The user enters a city name in the search field.
JavaScript sends a request to a weather API.
The API returns weather data in JSON format.
JavaScript processes the data.
The weather details are displayed dynamically on the webpage.

Project Structure :-

index.html – Contains the structure of the application
style.css – Contains styling and layout design
script.js – Contains API integration and logic

Learning Outcomes :-

Understanding API integration using JavaScript
Working with JSON data
Improving DOM manipulation skills
Enhancing responsive design knowledge
Strengthening debugging and problem-solving skills
* Conclusion :-
This Weather App project demonstrates practical implementation of front-end development concepts and API handling. It helped improve real-world development skills and understanding of dynamic web applications.
